### Но пока что самое важное, это как можно с помощью этого модуля безопасно сравнивать пароли

`
у нас бывает потребность сравнить пароли. Но сравнивать с помощью == не безопасно
потому что == проверяет по символьно, а так можно просчитывать, какая часть пароля угадана. Короче бред
`
``
чтобы такого не происходило, и мы проверяли пароль безопасно, на помощь нам приходит модуль secrets
``

```python
import secrets

a = "hello"
b = input()
print(secrets.compare_digest(a, b)) # вернет True or False
```
