# Ну тут немного про регулярки. Чтобы было куда подглядеть


`
Функция re.search() ищет первое совпадение шаблона. Если совпадение найдено, возвращает объект, иначе None
получается, чтобы получить строку, нужно потом воспользоваться .group()
`

```python
import re

text = "сегодня пятница, а завтра суббота"
result = re.search("пятница", text)

if result:
    print(result)
    print(result.group())  # Пятница


result = re.search("понедельник", text)
print(result)  # None

```

`
Функция re.findall() возвращает список всех совпадений шаблона
`

```python
import re

text = "У меня 2 пары носков, 0 рублей на карте, а вообще Я Никита! Мне 999 лет"

numbers = re.findall(r"\d+", text)

print(numbers)  # ['2', '0', '999']

```

## Специальные символы. Дай бог их запомнить
*  `.` - любой один символ, кроме символа переноса строки
*  `\d` - любая цифра от 0 до 9 включительно
*  `\w` - любая буква, цифра или подчеркивание
*  `\s` - любой пробельный символ (типа табуляция и тд)
*  `+` - один или более повторений предыдущего символа группы
*  `*` - ноль или более повторений
*  `?` - ноль или одно повторение
*  `[]` - набор символов. Например, `[абв]` означает любую группу из приведенных
*  `[^..]` - означает все символы, кроме перечисленных. Логическое не предыдущего 
*  `()` - группировка. Позволяет сохранять часть совпадения
*  `{n}` - показывает сколько раз символ или группа должна встречаться. Например, {3}, говорит, что символ или группа 
должны повториться 3 раза
*  `{n, m}` - от n до m повторений включительно

`
Ищем все почты текста
`

```python
import re

text = "Пишите нам на info@example.com или support@domain.org"
all_emails = re.findall(r"\w+@\w+.\w+", text)
print(all_emails)
```
`
Иногда требуется проверить соответствует ли строка формату
`
```python
import re


def is_phone_number(text):
    pattern = r"^\+7\s?\d{3}\s?\d{3}-\d{2}-\d{2}$"
    return re.match(pattern, text) is not None


print(is_phone_number("+7 123 456-78-90"))
print(is_phone_number("+7123456-78-90"))
print(is_phone_number("123-456-7890"))
```
``
в данном случае ^ - это означает начало строки, а $ - конец строки
``

## Функция re.sub()

``
заменяет все, найденные выражения шаблона
``

`
пример, где мы убираем все лишние пробелы. Оставляем только по 1
`
```python
import re

text = "Ооо,    всем привет! С вами       Влад        А4 гилент    кобяков"

normal_text = re.sub(r"\s+", " ", text)

print(normal_text)  # Ооо, всем привет! С вами Влад А4 гилент кобяков
```

`
Иногда нам нужно из, найденного совпадения, извлечь отдельные части
`

```python
import re

date_only = "03.05.2006"

group_date = re.search(r"(\d{2})\.(\d{2})\.(\d{4})", date_only)

if group_date:
    day = group_date.group(1)
    month = group_date.group(2)
    year = group_date.group(3)

    print(f"your pa was born in {day=}, {month=}, {year=}")  # your pa was born in day='03', month='05', year='2006'
```
